How do you create a makefile? Use a text editor. make files consist of rules. target, dependencies, and the recipe

Under what circumstances would you use cobol? Business applications. It natively works with decimal numbers.

Where is fortran a language of choice? For numerical computation (matrices but that goes under numerical computation). Where else would you use it (say, over cobol)? Fortran has native complex arithmetic.

Where is C the language of choice? When you need to manage memory. What kinda of program would you expect to be responsible for a lot of memory managment? Operating Systems(?). C was invented specifically for OPs systems. It needed to be small and portable to go on a lot of machine (it is relatively easily ported from machine to machine).

Why use subversion over git? When you want full control over your repo (this is one possibility but not strong enough). Under what circumstances would you use git? When you have unreliable communication. Subversion does very well when you have reliable communication but not so when you don't.

git = good for when you have unreliable communication
svn = good for when you have reliable communication

What's the next step after determining what the problem is? Solve it then code it. These are two separate steps. Be certain to specify this. You solve and then you program it.

What's a class? (in the context of OOP?) It is a blueprint (a specification) for an object.

What is global data? Only way to define this with an example (use java or c++). What is an element of global data? Data that can be accessed anywhere in the object. Fortran is guilty of this. Has something called common storage that is accessible throughout the entire program.

What's an instance of a class? An object. An instance of what is specified by the class. Remember, do not mix these up! Class is an entry in a catalogue. Object is what you order from an entry in the catalogue.

Why are global variables poison? If there is an error, it can be hard to find because it can come from any method. Therefore, it is poison.

What is the difference between decimal and floating point arithmetic and why is floating point treated as inaccurate? floating point is only an approximation. It can be very close but it's still an approximation. Error is effectively built in.

What is the difference between imperative and declarative programming? Imperative we have to tell the machine how to do something. Declarative we tell the machine what we want.

What is the definition of a function? In scheme, it is a set of arguments together with a rule for manipulating those arguments to obtain a simpler value. However, this varies widely within the context such as in calculus.

What is BNF? (Backus-Nauer Form) It is a language (technically not itself a programming language) for specification of context-free grammars, often used to describe the syntax of languages used in computing, such as computer programming languages, document formats, instruction sets, and communication protocols (partly his words and also a definition I found online).

What are the components of a scheme list? There are two. Head and tail.

Advantage of using recursion of loops such as while or for? Recursive functions can be easier to conceptualize (understand). Sometimes could be quicker than a for-loop. It's not always slower than a for loop or take up more memory. This isn't always the case. It'll be at least as fast as a for-loop.

What does define do in scheme? It binds names to values.

What were some errors in the therac-25? There was a performance issue. (also, there was a race condition if you were to enter information too quickly) If the machine were faster, it would work properly. It was less a software error and more of a performance error. There was, however, one absolutely clear software error. The answer is there was an arithmetic overflow. What happens if you increment a byte of size 256bits by 256 times? It overflows!

What is the purpose of making a trunk in subversion? It is merely a directory. You create directories to keep some order to your repository.

What is the fundamental difference between git and subversion? This is not the communication answer from before (although git was designed to help with teams that lack in communication). There is one huge difference between the two that makes everything else work. What is that big difference? (He says our silence says says we don't actually know what a git repo is despite using it so much). Git is distributed and subversion is centralized. Subversions repo is monolithic.

In scheme, what is car and cdr? car is the head and cdr is the tail which is an element.

In scheme, must all elements of the list be of the same type? The answer is no. They don't have to be.

What command is used to execute a make program? make.

What is one reason RCS was superseded by CVS? The amount of disc space became too much. RCS copied files repeatedly.

What does append do in scheme? Combines two lists by adding to the right side of the list. Works with two lists.
What about cons? Puts an element at the head of a list. At the left end. Cons works on an element and a list.

How many people were killed by therac-25? Six. The number is actually higher Gross' physician says but they didn't report it.

When do we use make (what is the general purpose of make?) It has an overarching single purpose that extends beyond programming alone. We use it to construct large things from small ones.
